HTML Básico

ATENÇÃO: tudo que em HTML é 'id' em CSS é '#'. Tudo que em HTML é 'class' em CSS é '.' (ponto). O meu primeiro h1 'criando sites com ttml e css' eu quero que seja um h1 especial, e que quando eu for fazer configuação ou rodar meu código ele seja h1 mas um pouco diferente. Por isso vou colocar um 'id' nele pra dizer que ele é importante e depois na minha folha de estilo eu vou fazer uma configuração específica só para esse h1. Nesse caso eu vou deixar meu título centralizado. Para escrever em duas classes ao mesmo tempo é só apertar 'alt' e clicar ond quero escever também

HTML Intermediário

>Dentro do mesmo documento HTML só posso ter um 'id' Eu posso personalizar varios elementos (h2, por ex) pra isso eu vou usar a 'class'. Lembra: 'id' é único 'class' pode ser multiplo.

HTML avançado

No mesmo texto posso ter uma classe e um id, mas o id vai sobrepor a classe sempre, se tiver fazendo a mesma coisa
