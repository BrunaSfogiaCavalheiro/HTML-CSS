Modelos de Caixas box-level

Paragrafos também são exemplos de box-level, mas os links são exemplos de caixas inline-level Vamos ver como tudo isso funciona

Para centralizar uma caixa é só usar 'margin: auto'

Para não ter que ficar com um style gigante tipo margin-top: 10px; margin-right: 10px; margin-bottom: 10px; margin-left: 10px; eu posso colocar na sequencia > margin: 10px 10px 10px 10px, sempre no sentido horário. E como esses numeros são iguais basta eu colocar só uma vez que já vale pra todos os lados, então pode ficar: margin 10px; e pronto.

A função 'display' pode definir se vai ser 'block level ou inline-level'. Não fiz isso aqui, mas é uma funcção que posso usar. O 'DISPLAY' MUDA A CARACTERISTICA NORMAL DE UM ELEMENTO
